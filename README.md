### Приложение Stellar Burgers. Это космический фастфуд: можно собрать и заказать бургер из необычных ингредиентов..
### Подключен Selenium
### Написана функция, которая генерирует логины(имя, email, пароль). Подключен генератор логина к тестам.
### Создана функция для авторизации и входа в файле helper.py, чтобы минимизировать повторения в коде
### Оптимизированы все локатор и вынесены в отдельный файл locators.py


### Списoк реализованных тестов 
1. Регистрация нового пользователя.
2. Проверка, что появляется текстовое сообщение "Некорректный пароль", при введении некорректного пароля, состоящего из 5 символов.
3. Проверка регистрации нового юзера с пустым полем "пароль"
4. Проверка регистрации нового юзера с пустым полем "имя"
5. Проверка входа по кнопке "Войти в аккаунт" на главной
6. Проверка входа через кнопку "Личный кабинет" 
7. Проверка входа через кнопку в форме регистрации
8. Проверка входа через кнопку в форме восстановления пароля
9. Проверка перехода по клику на личный кабинет
10. Проверка перехода по клику на конструктор
11. Проверка перехода по клику на логотип Stellar Burgers
12. Проверка выхода по кнопке "Выйти" в личном кабинете
13. Проверка, что работают переходы к разделам: "Булки", "Соусы", "Начинки"
